# ROS 2 Educational Book

This is an educational book about ROS 2 (Robot Operating System 2) architecture, focusing on core concepts for humanoid robot control and AI integration. The content is designed for AI and robotics students with basic Python knowledge.

## About

This book covers essential ROS 2 components including nodes, topics, services, communication models, Python agents with rclpy, and URDF fundamentals for humanoid robot modeling. The content focuses on conceptual understanding rather than hands-on implementation to match the target audience of students with basic Python knowledge.

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

## Build

```bash
# Build for production
npm run build
```

## Features

- Educational content about ROS 2 architecture
- Three main chapters: ROS 2 Architecture, Python Agents with rclpy, and URDF for Humanoids
- Clear, instructional tone appropriate for students
- Minimal code examples to illustrate concepts
